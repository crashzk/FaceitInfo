# FaceitInfo
Модуль для внедрения статистики Faceit во Flute. Автоматически добавляет блок о Faceit у пользователя, а так же позволяет другим модулям использовать Faceit.  Module for embedding Faceit statistics into Flute. Automatically adds a Faceit block to the user, and also allows other modules to use Faceit.
